# Audio Spectrum Designer

An interactive Web Audio lab for designing tones and analyzing their frequency-domain profile in real time.

## Highlights
- Oscillator waveform selection and live synthesis controls.
- LFO-based frequency modulation.
- Real-time FFT spectrum visualization on Canvas.
- Signal metrics: peak frequency, spectral centroid, and energy.

## Stack
- Web Audio API
- HTML5 Canvas
- Vanilla JavaScript + CSS dashboard

## Run
Open `index.html` and click **Start Tone** (browser gesture required for audio).
